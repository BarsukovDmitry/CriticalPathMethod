    2.4. Экспорт в MS Project

После расчёта ускорения продолжить работу над проектом можно в MS Project. Для этого требуется выполнить команду Файл->Экспорт->Проект для MS Project. Затем данный проект открывается в MS Project. Для этого в окне открытия файла выбирается тип файла "текстовые файлы". Далее, следуя инструкциям мастера и оставляя настройки по умолчанию, вы создаёте проект: Далее->Создать новую схему->Создать новый проект->Задачи->Далее_>Готово.  В результате получается:

Как видно, все работы были перенесены в MS Project (интерфейс немного изменён для более полного отображения соответствия результатов вычислений в MCP и MS Project).
