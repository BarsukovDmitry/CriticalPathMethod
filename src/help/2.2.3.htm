    2.3. Расчёт

   

Теперь можно приступать к расчётам. Для этого щёлкните Проект->Рассчитать. Получается следующее:

 

    Все расчёты для нормального срока проекта выполнены. Для расчёта ускорения на панели инструментов "Объекты" введите ускорение. По условию задачи - 4 дня. И снова рассчитайте проект. Опционально можно включить Проект->Уведомления при расчёте.

 

    Ну и опять же опционально завершить можно отчётом в html. Для этого выберите Файл->Экспорт->Отчёт о проекте (html)...
